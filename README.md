# Hacknation

## Techstack

+ Backend : fastapi
+ Frontend : Vite + React
+ UI lib : Shadcn

## To get most out of the dev experience

1. Set up 
```bash
brew install mise
brew install mprocs
mise trust
mise install
mise run i
cd front && npm install
```

2. To start both backend and frontend run `mprocs`

3. Check frontend at http://localhost:5173 and backend at http://localhost:8000/docs
